## Local Development

Prerequisites:
- Node.js v20+
- pnpm v9+

Installation steps:

1. Clone repository:
```bash
git clone https://github.com/CodeF1ow/codextreme-web.git
cd codextreme-web
```
2. Install dependencies:
```bash
pnpm install
```
3. Start development server:
```bash
pnpm run dev
```
4. Build for production:
```bash
pnpm run build
```
